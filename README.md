# Car Showroom Management System

## Description
This is a Python-based Car Showroom Management System developed as a school project.
It allows users to add, search, update, delete, and display car records using file handling.

## Features
- Add new car records
- Search car details
- Update car information
- Delete car records
- Display all car details

## Technologies Used
- Python
- File Handling

## How to Run
1. Install Python on your system
2. Run the program using:
   python main.py
